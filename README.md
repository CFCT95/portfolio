# Cristian Torres — Portfolio

Personal portfolio website built with Next.js 14 + Tailwind CSS, deployed on Vercel.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project → select this repo
3. Vercel auto-detects Next.js — click Deploy
4. Every `git push` auto-deploys

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Hosting:** Vercel (free tier)
